# **App Name**: Ehsan study website login page

## Core Features:

- Email Authentication: Authenticate users via email and password using Firebase Authentication.
- Form Validation: Validate email and password inputs with real-time error handling.
- User Redirection: Redirect authenticated users to https://ehsan.42web.io/ESW/ upon successful login.
- Error Handling: Display specific error messages for invalid email or incorrect password.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to evoke trust and serenity.
- Background color: Very light indigo (#E8EAF6) as a calming backdrop.
- Accent color: Vibrant purple (#7E57C2) for interactive elements.
- Body and headline font: 'Inter' sans-serif, for a modern, clean, transparent, and readable aesthetic.
- Glass-transparent login box centered on a full-screen background image. The layout is designed to be fully responsive across desktop and mobile.
- Smooth, subtle transitions and animations on form elements.
